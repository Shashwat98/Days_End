# Days_End

## Requirements
- Unreal Engine [your version]
- Visual Studio 2022

## Setup Instructions
1. Clone this repository
2. Download the Content assets from Google Drive: (https://drive.google.com/drive/folders/14g0FoaqUtW1IuaM2NN2DoiSD5UfmeBmc?usp=sharing)
3. Extract and place all downloaded folders inside the `Content/` folder
4. Right-click `Days_End.uproject` → Generate Visual Studio project files
5. Open `Days_End.uproject` and let Unreal rebuild
